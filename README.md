## 📌 Overview

**Nifty Options Assist PRO** is a custom-built, multi-confirmation 
trading indicator developed in Pine Script on TradingView. 

The system combines **10+ technical indicators** into a single 
unified engine that analyzes market structure, momentum, volatility, 
and volume — and outputs a clear **BUY CE** (Bullish) or 
**BUY PE** (Bearish) signal for Nifty 50 options trading on NSE.

The core philosophy behind this system is **confluence** — a trade 
signal is only generated when the majority of indicators agree, 
significantly reducing false signals and improving accuracy.

> Built and actively used for intraday and positional 
> Nifty 50 options trading.

---

## 🧠 System Architecture

The indicator runs a **Bull Score Engine** in the background that 
assigns a score to current market conditions based on all active 
indicators. The final BIAS (CE or PE) is determined by this score.
